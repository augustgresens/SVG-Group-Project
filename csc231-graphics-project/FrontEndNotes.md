# Customer Request:

- "I want to make an image of a green rectangle."
- "I want a background of triangles, with ever other line inverted."
- "I want to make a pattern of circles that spirals out on a cartesian grid (e.g., 0,0 1,3 2,6 etc.)" 

# Classes:
- Base Class:
    - Shapes()
- Dependent Classes:
    - Triangle()
        - Data:
            - 3 points
            - Line Color
            - Fill Color
    - Parallelogram()
        - Data:
            - 4 points
            - Line Color
            - Fill Color
    - Circle()
        - Data:
            - center point
            - radius
            - Line Color
            - Fill Color

# Patterning
- moveto(x,y)
- shift(dx,dy)
- rotate(angle)
